### 调试主进程
使用如下的命令行开关来调试 Electron 的主进程：

- --inspect=[port]  
当这个开关用于 Electron 时，它将会监听 V8 引擎中有关 port 的调试器协议信息。 默认的port 是 5858

```electron --inspect=5858 your/app```

- --inspect-brk=[port]  
和`--inspector` 一样，但是会在JavaScript 脚本的第一行暂停运行。
